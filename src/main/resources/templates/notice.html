공지사항

해당 홈페이지는 실제로 사용할 수 있는 홈페이지가 아닌 개인 학습용 프로젝트로 제작한 홈페이지입니다.

제작할 때 참고한 실제 쇼핑몰 홈페이지는 http://prettypop.co.kr 입니다.

실제 운영되는 사이트가 아니므로 회원가입을 할 때 실제 개인정보가 아닌 임의의 데이터를 입력하여 사이트를 이용해주세요.
결제는 구현하지 않았기 때문에 회원가입 시 지급되는 포인트로만 상품 주문을 진행할 수 있습니다.
상품을 주문하면 바로 주문 완료, 배송 완료 상태가 됩니다.

인원 : 1인
개발 : Spring Boot, Thymeleaf, JPA, Spring Data JPA, QueryDSL
깃허브 : https://github.com/h-spear/pretty-pop
